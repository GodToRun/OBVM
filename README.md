# OBVM
OBVM is a Object-Based Virtual Machine.
and OBVM is open source!

This VM writed with Java.

# Principle of Operation
OBVM uses independent D-DOS and inserts objects using ArrayList in the file system. And OBVM separates files by indexes.

It's an object-based virtual machine, so be careful that once you exit the program, everything is gone.
